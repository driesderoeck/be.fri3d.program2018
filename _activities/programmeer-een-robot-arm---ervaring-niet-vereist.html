Een leuke programmeer workshop met de uArm pro robot (een 5-axis robot arm).

De uArm robot kan via een grafische (scratch) omgeving voor beginners of met Python voor gevorderden geprogrammeerd worden.

Voor kids vanaf ~8 jaar (en natuurlijk voor iedere leeftijd daarboven)
