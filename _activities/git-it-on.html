Een korte introductie tot git, een bekend versie beheersysteem. Tijdens deze kennismaking leer je het principe van commits, branches aan de hand van real-life voorbeelden.

Deelnemers hebben de mogelijkheid om achteraf een initiële ervaring op te doen met git a.d.h.v. een paar praktijkoefeningen.
